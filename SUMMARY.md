## 💡 Key Insights

### 🔴 Nguyên Nhân Lag Chính:
1. **Bilibili body rewrite** (5000+ ký tự JQ query) - NẶNG NHẤT
2. 45+ scripts phải check mỗi request
3. 40+ MITM hostnames
4. Unlimited body size processing
5. 120s timeout

### ✅ Giải Pháp:
1. **Loại bỏ Bilibili** → Giảm 70% lag
2. **Giảm scripts** 45 → 11 (Balanced) hoặc 2 (Ultra Fast)
3. **Giảm MITM hosts** 40 → 10 hoặc 3
4. **Limit body size** unlimited → 3MB
5. **Giảm timeout** 120s → 10-30s

### 📈 Kết Quả:
- Load time: 2.5s → 1.2s (Balanced) hoặc 0.8s (Ultra Fast)
- Battery: -15% → -8% hoặc -5%
- CPU: 25-40% → 8-15% hoặc 5-10%
- Lag: 100% → <10% hoặc 0%
