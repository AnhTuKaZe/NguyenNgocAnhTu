### 1Ô∏è‚É£ Surge (iOS / macOS)

<details>
<summary><b>Step-by-step guide</b></summary>

<br>

**Import Module**
1. Open **Surge** ‚Üí **Home** tab
2. Tap **Modules** ‚Üí **+** button (top right)
3. Select **Install Module from URL**
4. Paste URL ‚Üí **OK**
5. Module appears in list

**Install Certificate**
1. **Settings** ‚Üí **MITM** ‚Üí **Install Certificate**
2. Safari opens ‚Üí Install profile
3. Enter passcode ‚Üí Install
4. **Settings** ‚Üí **General** ‚Üí **About** ‚Üí **Certificate Trust Settings**
5. Enable trust for **Surge** certificate

**Verify**
- Module enabled (green toggle)
- MITM enabled
- Test apps
